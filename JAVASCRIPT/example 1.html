1. JavaScript Basics for Web Development

JavaScript is a language which is used to make the websites interactive and working with the user. It is a core technology of the World Wide Web, alongside HTML and CSS. JavaScript enables dynamic content, control multimedia, animate images, and much more.

  • HTML → Car ka body
  • CSS → Car ka paint & design
  • JavaScript → Engine jo car chalata hai

2. Two ways of write the JavaScript

There are two ways to write JavaScript code in a web page: inline and external.

1. Inline JavaScript

Inline JavaScript is written directly within the HTML elements using the onclick attribute or other event attributes.

3. Variables – Data Store karne ka Box

Variables are used to store data values. In JavaScript, you can declare variables using var, let, or const.

  • Jab value change hone wali hai → let
  • Jab value fix hai → const

4. Data Types – Data ka Type (Kis type ka data store ho raha hai)

JavaScript has different data types to represent various kinds of data. The main data types are:

  • String: Textual data
  • Number: Numeric data
  • Boolean: True or false values
  • Null: empty value intentionally
  • Object: Complex data structures or Key Value
  • Array: Ordered list of values

5. Type Conversion - Change the Data Type

Type conversion is the process of converting a value from one data type to another. JavaScript can also explicitly convert types using functions like String() and Number().
