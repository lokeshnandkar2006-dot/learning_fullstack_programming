Local Storage / Session Storage

Local Storage and Session Storage are web storage objects that allow you to store data in the browser.

Local Storage

Local Storage allows you to store data with no expiration date. The data will not be deleted when the browser is closed and will be available the next day, week, or year.

Session Storage

Session Storage is similar to Local Storage, but the data is deleted when the browser tab is closed.

Key Differences

  • Local Storage data persists until explicitly deleted.
  • Session Storage data is deleted when the tab is closed.
  • Both can store up to 5-10MB of data depending on the browser.
