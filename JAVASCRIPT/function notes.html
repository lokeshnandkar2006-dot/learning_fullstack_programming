Functions, Function Types & Scope

FUNCTIONS

A function is a block of code designed to perform a particular task. It is executed when "something" invokes it (calls it).

Think of a function like: “Ek employee jo ek kaam karta hai jab tum use bulaate ho.”

FUNCTION TYPES

1. Function Declaration

  • function greet() → function define kiya
  • {} → iske andar likha code jab function call hoga tab chalega
  • greet() → function ko call kiya

2. Function Expression

  • const sayHello = → function ko variable me store kiya
  • function () → function define kiya (iska koi naam nahi hai)
  • {} → iske andar likha code jab function call hoga tab chalega
  • sayHello() → function ko call kiya

3. Parameters & Arguments

  • a, b → parameters (placeholders)
  • 5, 10 → arguments (actual values)
  • {} → iske andar likha code jab function call hoga tab chalega
  • add(5, 10) → function ko call kiya

4. Return Statement

  • x, y → parameters (placeholders)
  • 4, 6 → arguments (actual values)
  • { return x * y; } → function ka result wapas karta hai
  • const result = → function ke return value ko variable me store kiya
  • multiply(4, 6) → function ko call kiya

5. Arrow Functions (Modern JS)

  • const divide = → function ko variable me store kiya
  • (m, n) → parameters (placeholders)
  • { return m / n; } → function ka result wapas karta hai
  • const divisionResult = → function ke return value ko variable me store kiya
  • divide(20, 4) → function ko call kiya

6. Function Hoisting

Tum function ko uske define hone se pehle bhi call kar sakte ho — if it is a function declaration.

  • hoistedFunction() → function ko call kiya (define hone se pehle)
  • function hoistedFunction() → function define kiya
  • {} → iske andar likha code jab function call hoga tab chalega

8. Default Parameters

  • name = "Guest" → default parameter value set kiya
  • {} → iske andar likha code jab function call hoga tab chalega
  • greetUser() → function ko call kiya (default value use ki)
  • greetUser("Alice") → function ko call kiya (custom value use ki)

9. SCOPE (Global vs Local)

Scope determines the accessibility (visibility) of variables. JavaScript has two types of scope: Global Scope and Local Scope.

i. Global Scope

  • globalVar → globally accessible variable
  • showGlobalVar() → function jo global variable ko access karta hai

ii. Local Scope

  • localVar → sirf function ke andar accessible variable
  • showLocalVar() → function jo local variable ko access karta hai

10. Block Scope (let & const)

  • blockVar → sirf block ke andar accessible variable
  • { } → block jisme variable define kiya gaya hai
