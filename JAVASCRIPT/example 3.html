JavaScript Day 2        

Conditions + Loops (The Logic Behind All Websites)

   

      Conditions and loops are fundamental concepts in programming that allow       developers to control the flow of their code based on certain criteria and       repeat actions multiple times.    

   
   

1. Conditional Statements

   

      Conditional statements allow you to execute different blocks of code based       on whether a specified condition is true or false. The most common       conditional statements in JavaScript are if / else,       else-if, and switch..    

   
   

IF Statement

   

      The if statement executes a block of code if a specified       condition is true.    

       
   

IF...ELSE Statement

   

      The if...else statement executes one block of code if a       specified condition is true, and another block if it is false.    

       
   

IF...ELSE..IF Statement

   

      The if...else if...else statement allows you to test multiple       conditions.    

       
   

NESTED CONDITIONS

   

      Nested conditions are conditional statements placed inside other       conditional statements. This allows for more complex decision-making       processes in your code.    

       
   

TERNARY OPERATOR

   

      The ternary operator is a shorthand way of writing an       if...else       statement. It takes three operands: a condition, a value if the condition       is true, and a value if the condition is false.    

       
   

SWITCH Statement

   

      The switch statement is used to perform different actions       based on different conditions. It is often used as an alternative to       multiple if...else if statements.    

       
   

2. LOOPS (for, while, do-while, for-of, for-in)

   

      Loops are used to repeat a block of code multiple times until a certain       condition is met. JavaScript provides several types of loops, including       for, while, do-while,       for-of, and for-in.    

   
   

FOR Loop

   

      The for loop is used to repeat a block of code a specific       number of times.    

       
   

WHILE Loop

   

      The while loop repeats a block of code as long as a specified       condition is true.    

       
   

DO-WHILE Loop

   

      The do-while loop is similar to the while loop,       but it guarantees that the block of code will be executed at least once       before checking the condition.    

       
   

FOR-OF Loop

   

      The for-of loop is used to iterate over iterable objects       (like arrays, strings, etc.).    

       
   

FOR-IN Loop

   

      The for-in loop is used to iterate over the properties of an       object.    

       
   

BREAK + CONTINUE

   

      The break and continue statements are used to       control the flow of loops.
