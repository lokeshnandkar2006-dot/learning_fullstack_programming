1. CSS Selectors (class, id, element)

CSS selectors are used to select the HTML elements you want to style.

  • Element selector → selects all elements of a type
  • Class selector (.) → used for multiple elements
  • ID selector (#) → used for one unique element

Important Tags:

  • <div> - Block-level container. It is used to group elements together. Act like a box.
  • <span> - Inline container. Used to style small parts of text. Unlike <div> tag, it does not start on a new line (inline element).

2. Box Model (margin, padding, border)

The CSS box model consists of: Content, Padding, Border, and Margin.

  • Content: The actual content of the box, where text and images appear.
  • Padding → space inside border
  • Border → line around the element
  • Margin → space outside the border

3. Colors and Backgrounds

CSS allows you to set colors and backgrounds for elements.

Ways to apply colors:

  • Color names → red
  • Hex → #ff0000
  • RGB → rgb(255,0,0)

Background properties:

  • background-color
  • background-image
  • background-size, etc.
